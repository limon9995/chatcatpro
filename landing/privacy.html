Privacy Policy

এই Policy-তে আমরা ব্যাখ্যা করেছি Chatcat কীভাবে আপনার তথ্য সংগ্রহ, ব্যবহার এবং সুরক্ষিত রাখে।

1. আমরা কারা (Who We Are)

Chatcat একটি SaaS platform যা Bangladeshi Facebook sellers দের জন্য order automation, courier integration, CRM ও accounting সেবা প্রদান করে। আমাদের service chatcat.pro এবং app.chatcat.pro দিয়ে পাওয়া যায়।

যোগাযোগ: info@chatcat.pro

2. আমরা কী তথ্য সংগ্রহ করি (Information We Collect)

আমরা নিম্নলিখিত তথ্য সংগ্রহ করি:

  • Account information: আপনার নাম, email address, password (encrypted)
  • Facebook Page data: Page ID, Page access token (AES-256 encrypted), Page name — আপনি যখন আপনার Facebook Page connect করেন
  • Customer data (আপনার customers-এর): Facebook PSID (Messenger user ID), নাম, ফোন নম্বর, ঠিকানা — order process করার সময় customers যা দেন
  • Order data: Product code, quantity, delivery address, courier tracking info
  • Conversation data: Bot conversation context (short-term, order flow-এর জন্য)
  • Usage data: API usage logs, AI token usage (billing-এর জন্য)
  • Payment data: Subscription status — actual payment আমরা process করি না, third-party gateway দিয়ে হয়

3. তথ্য কীভাবে ব্যবহার করি (How We Use Your Information)

সংগৃহীত তথ্য শুধুমাত্র নিম্নলিখিত কাজে ব্যবহার হয়:

  • আপনার Chatcat account পরিচালনা করতে
  • Facebook Messenger bot-এর মাধ্যমে order নিতে ও customer-দের reply দিতে
  • Courier booking (Pathao, Steadfast, RedX, Paperfly) সম্পন্ন করতে
  • Dashboard-এ order, CRM ও accounting দেখাতে
  • AI features (OCR, intent detection) চালাতে
  • Billing ও subscription management করতে
  • Support ও product improvement-এর জন্য (anonymized analytics)

4. Facebook Data ও Meta Platform Policy

Chatcat Facebook Messenger Platform ব্যবহার করে। এতে আমরা Meta-র Platform Policy এবং Data Policy মেনে চলি।

  • আমরা শুধু সেই data access করি যা bot operation-এর জন্য প্রয়োজন
  • Facebook PSID তৃতীয় পক্ষের সাথে share করা হয় না
  • Customer-এর Messenger conversation data তৃতীয় পক্ষের সাথে বিক্রি বা share করা হয় না
  • আপনার Facebook Page access token AES-256 encryption দিয়ে সুরক্ষিত রাখা হয়

Facebook account disconnect করলে আমরা সংশ্লিষ্ট Page token মুছে ফেলি।

5. Data সুরক্ষা (Data Security)

  • সমস্ত sensitive data (tokens, passwords) AES-256 encryption-এ store করা হয়
  • HTTPS/TLS দিয়ে সমস্ত connection encrypted
  • Facebook webhook আসলে HMAC-SHA256 signature verification করা হয়
  • Dashboard access role-based authentication দিয়ে সুরক্ষিত
  • Rate limiting দিয়ে brute-force attack প্রতিরোধ করা হয়

6. তৃতীয় পক্ষের সাথে Data Sharing

আমরা আপনার personal data কোনো বিজ্ঞাপনদাতার কাছে বিক্রি করি না। নিম্নলিখিত third parties-এর সাথে শুধুমাত্র service delivery-এর জন্য data share করা হয়:

  • Meta (Facebook): Messenger API দিয়ে message send/receive
  • OpenAI: AI reply generation-এর জন্য (message text — no personal identifiers)
  • Courier partners: Pathao, Steadfast, RedX, Paperfly — order delivery info
  • Hosting provider: Server infrastructure

7. Data Retention (তথ্য কতদিন রাখা হয়)

  • Account data: account সক্রিয় থাকা পর্যন্ত
  • Order data: ব্যবসায়িক record-এর জন্য minimum 2 বছর
  • Conversation context: ৩০ দিন পর automatically expire
  • Account delete করলে: ৩০ দিনের মধ্যে সমস্ত personal data মুছে ফেলা হয়

8. আপনার অধিকার (Your Rights)

আপনার নিম্নলিখিত অধিকার আছে:

  • আপনার সম্পর্কে কী data আছে জানার অধিকার
  • ভুল data সংশোধনের অধিকার
  • Data deletion-এর অধিকার (account delete করুন বা email করুন)
  • Data portability — আপনার order data CSV-তে export করতে পারবেন

যেকোনো request-এর জন্য: info@chatcat.pro

9. Cookies

আমরা শুধুমাত্র essential cookies ব্যবহার করি — authentication session বজায় রাখতে। কোনো advertising বা tracking cookie ব্যবহার করা হয় না।

10. Children's Privacy
